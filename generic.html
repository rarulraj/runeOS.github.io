Kernel

runeOS has a microkernel architecture, where most of the operating system services run in user space, allowing for better isolation between components, which enhances security and stability.

The kernel provides only the basic services needed for hardware management and communication between processes, ensuring high performance, low resource usage, and improved security.

To create runeOS, we started with a minimal kernel and built on top of it carefully. The kernel is designed in such a way that additional services and features can be added without compromising its minimal nature. The goal is to create an operating system that is simple, reliable, and fast, with minimal dependencies and low overhead.
