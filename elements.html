TIPS

Accesibility

Familiarize yourself with the command-line interface: Since RuneOS has a command-line interface as the default interface, it's important to become comfortable with using the command-line. You can start by learning basic commands, such as cd (change directory), ls (list files), and mkdir (make directory). You can also use the --help option to get more information about a specific command.

More Info

Customize your environment: RuneOS is designed to be simple and minimal, but you can still customize your environment to suit your preferences. You can modify your shell prompt, set environment variables, and install additional software as needed.

Tools

Use Rust-based tools: Since RuneOS is built using Rust, it's a good idea to use Rust-based tools when possible. For example, you can use the Rust-based text editor, file manager, and terminal emulator that come with RuneOS.
