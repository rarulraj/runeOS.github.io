About rune

Welcome to rune OS
work smarter and more efficent

rune OS is built with RUST

The OS comes with a set of basic tools, including a text editor, a file manager, and a terminal emulator, all written in Rust to ensure speed and reliability.

Why rune?

While RuneOS is not suitable for running complex applications or workloads that require a more full-featured operating system, it is perfect for use in embedded systems, IoT devices, and other applications where simplicity and reliability are crucial.
